# 🔍 Port Scanner

Профессиональный сканер портов на Python для аудита безопасности.

## 📁 Файлы проекта

| Файл | Описание |
|------|----------|
| `scanner.py` | Базовая версия (однопоточная, учебная) |
| `scanner_advanced.py` | Профессиональная версия (многопоточная, определение сервисов) |

## 🚀 Возможности

- ⚡ Многопоточность (100 потоков)
- 🎯 Определение сервисов (SSH, HTTP, SMB, RPC и др.)
- 📄 Сохранение результатов в файл
- 🎨 Цветной вывод

## 📝 Использование

```bash
python3 scanner_advanced.py 127.0.0.1
python3 scanner_advanced.py scanme.nmap.org
